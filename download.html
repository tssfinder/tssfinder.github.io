  • First release - v1.0.0
  • TSSFinder is implemented using Python V.3.6 and a customized compiled C++ extension of the ToPS probabilistic framework, and consists of two scripts: the training script and the prediction script.
  • TSSFinder is available for command-line execution for unix-based operating systems.
  • Six models available: A. thaliana, O. sativa, D. melanogaster, H. sapiens, S. cerevisiae, G. gallus
