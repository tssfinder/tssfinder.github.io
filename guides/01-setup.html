Setup

First you will need to download the last version of MYOP software. It includes:

  • MYOP command line tools
  • ToPS command line tools
  • Some pre-trained gene models

Install MYOP for Mac OS X

  1. Get the OS X installer here
  2. Doble-click the downloaded package. It will open a window like the following:
  3. Drag and drop the MYOP icon to the Applications folder.

After installing, check your installation:

  1. Go to you Applications folder
  2. Double-click MYOP icon. It will open a Terminal window.
  3. Type myop and hit <return>. It should list all MYOP’s commands and all available genomes and transcriptomes.

Install MYOP for Ubuntu Linux

  1. Get the .deb package here
  2. Open a terminal window and type dpkg -i myop-v1.0.0.deb

After installing, check your installation:

  1. Open a terminal window
  2. Type myop and hit <return>. It should list all MYOP’s commands and all available genomes and transcriptomes.

Building from Source

Firt install all the following requirements:

  1. ToPS
  2. SegSeq
  3. Git
  4. Perl - ForkManager
  5. BioPerl

After you have installed the required software, you can install MYOP by following the instructions below:

  1. MYOP is available from our GitHub repository.

    git clone https://github.com/myopdev/myop.git

  2. Copy the myop folder to /usr/local/ folder.

    sudo cp -r myop /usr/local

  3. Add the following line at the end of the .profile file.

    export PATH=$PATH:/usr/local/myop/scripts
